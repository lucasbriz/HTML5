Módulo 9 - Usando imagens mapeadas

Imagem mapeada é uma imagem dividida em diferentes áres e cada área é interligada a um documento. Elas podem ser tanto back como front-end. Para mapear as imagens é necessário pensar nela como uma forma geométrica
